    Work

    Here is a carefully chosen selection of projects I have worked on that reflect the state of my abilities so far using HTML, CSS, and JavaScript for front-end Web development.

    First project done during Bootcamp. Using server side APIs it allows the user to search movies by genere, look streaming availability, and concoct a list of favorites.

    Second project done during Bootcamp. Using data from the National Park Systems API, we let users search parks by state, by activities, and by topics and allow them to create lists of favorites, "been-there", and "bucket list" parks. In the non-membership section, all the data is set to localStorage while the membership section employs a MySQL database and lets fellow explorers communicate in a CMS blog.

    An App that allows the user to effortlessly genrate a random password of the desired length and with user-given character types.

    An App that helps the user to review basic notions of git Bash, HTML, CSS, and Javascript by taking a timed quiz.

    An App that helps the user to organize his/her working schedule, keeping track of what had to be done, what has to be done, and what will have to be done.

    An App that helps the user to decide what city to visit and what to do during each day of the visit based on current and futures weather conditions.
